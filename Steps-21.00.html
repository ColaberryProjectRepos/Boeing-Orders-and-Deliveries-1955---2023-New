Creating Cover Page 21

  1. Towards the bottom of the Dashboard there should be a section where it shows how many pages are within the project. 
  2. Press the green button with a plus sign to create a new page for the cover photo
  3. Once you are on page 2, you can now put the cover photo in the page so that we can use this as our cover photo when recording for a gif
  4. Head to the Insert tab and press the button Image
  5. Within your computer, open the zip file you downloaded from the beginning of the step-by-step instruction. 
  6. There should be a file named Boeing Project Cover Photo, select it and press ok
  7. Once the image loads into the dashboard, resize it so it fits the entire visual.
