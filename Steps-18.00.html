Creating YoY Arrows 18


This step will be to create an arrow by using the right measure to portray the correct information. 
  1. Double click on the Card icon on the visualization pane 
  2. A blank Card visual will appear on the report.
  3. For Order YoY %, you will need to drag and drop the column “Order YoY Arrow” in the Measures Table to the blank Card
  4. Once you have done that a black arrow should appear, now you will be able to make changes to the Card.
    1. Visual: 
      1.  Font: DIN 
      2. Text Size: 13
      3. For the Color, press the Fx button to make the color change based off the percentage. Once Pressed a Color - Callout Value window will appear. For the Format Style, “Field Value” should be selected. For the question, What field should we base this on?, you will have to select the Order YoY % Color column within the measures table.
      5. Once finished press the OK button and the arrow should change to a red or green color.
      6. Display Units: Auto
      7. Value Decimal Places: Auto
      8. Text Wrap: ON
      9. Source Spacing: ON
      10. Category Label: OFF 
    2. General:
      1. Height: 32
      2. Width: 32
      3. Horizontal: 582
      4. Vertical 144
      5. Title: OFF
      6. Effects -> Background: ON
      7. Color: #FFFFFF
      8. Header Icons: ON
      9. Background Color: #FFFFFF
      10. Border Color: #FFFFFF
      11. Icon Color: #333333
  5. Repeat Steps 1-4 to create the Delivery YoY Arrow as well, you should have something like this.
