Creating Column Stacked Charts 19
  1. Double click on the “Line and Stacked Column Chart” in the visualization pane.
  2. A blank report will appear waiting for you to add columns to create a visual. To create a visual you will need to add the following.
    1. X – Axis: Year (Groups)
    2. Column Y – Axis:
      1. Average Delivery
      2. Undelivered Orders
    3. Line Y – Axis: 
      1. Order YoY %
      2. Delivery YoY %
    4. Tooltips: 
      1.  Total Orders 
      2. Orders Per Month
      3. Order YoY %
      4. Delivery %
      5. Delivery YoY %
