  9.  Copy the Total Orders Card to replicate the 5 other KPI Cards you will have to make. In order to do this, hold the CTRL + C keys while the Total Orders card is selected. Once you have done this now use the CTRL + V to have a duplicate made of that card. Once a duplicate has been made, you are now able to make your changes to the new card. The changes that will have to be made are changing which data is shown in the card itself, the titles of each card, and finally the color of the Callout Value. You will create 5 additional cards with the following criteria:
    1. Orders Per Month will have a Purple (#744EC2) Callout Value
    2. Order YoY % will have a Light Blue (#007BFF) Callout Value 
    3. Delivery % will have a Black (#000000) Callout Value 
    4. Delivery YoY % will have a Orange (#E66C37) Callout Value
    5. Data Refresh will have a Black (#000000) Callout Value
