Transforming the Data (Renaming Engine & Getting Rid of Nulls in Unfilled Orders) 9

Still within the transform window, we will remove nulls from unfilled orders and change engine names since they came in the dataset as abbreviations.

1. Still in the transform window, right click the Unfilled Orders column
2. A Dropdown menu should appear, click on Replace Values




3. Once clicked, a window should appear to enter what information we want to change. For the Value to Find input the text null, for Replace With input 0 and click ok




4. Now we will have to change the abbreviations for the engine. Right Click the Engine column.
5. A dropdown menu should appear, click on Replace Values
6. Once clicked, a window should appear to enter what information we want to change. This time we will be changing the abbreviation to the actual name of the engine for the next 6 steps since there are 6 different engine types. 
            a. GE should be replaced with General Electric 
            b. PW should be replaced with Pratt & Whitney
            c. CF should be replaced with CMF International 
            d. RR should be replaced with Rolls-Royce
            e. NS should be replaced with New Surplus
            f. BR should be replaced with Bentley
