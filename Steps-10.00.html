Data Refresh 10

Once you have completed cleaning up the data within the power query editor, we will now create a table to display the data refresh time whenever the visual is refreshed. 

1. Click Transform Data in the Home tab in the ribbon.
2. Once the Power Query Editor window opens, you will need to click the New Source -> Blank Query 



3. Once you hit ENTER, you will now be able to type the DAX for the Data Refresh. Type in the box =DateTime.LocalNow() and hit enter


Once you hit ENTER, the query result should show the exact time and the ribbon will change to an orange color like this


4. Now we have the local date time from the local machine. We are able to create the table for the data refresh. In order to do this, you will have to click in the top left corner To Table 


5. Once you have changed the query result to a table, we will need to name our new table. Head to the right of the screen to Properties and change the name from query 1 to Data Refresh



6. Now that we have our table named and query brought over, you will have to change the heading of the first column and rename it. Right click the column and click on the Rename option in the drop down. Rename it Date Time
7. Once you have renamed the column, you will have to duplicate it twice. Right Click the Date Time column to get to the drop down and click the Duplicate Column option, do this step twice.
8. Now for the first duplicated column, rename it Date and for the other duplicated column rename it Time 
9. For the Date column, right click the header and click the Change Type -> Date
10. For the Time column, right click the header and click the Change Type -> Time
