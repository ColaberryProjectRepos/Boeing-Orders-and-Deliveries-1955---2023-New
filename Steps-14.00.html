Implementing Measures 14

In this step we will be adding all the measures in order for us to us within our visual.
 1. Click on Transform Data to open up the Power Query Window 

2. Once open click the Enter Data button in the ribbon to create a new table ?


3. Once clicked, a Create Table window will open and prompt you to put a title towards the bottom. You will put the title as Measure and once youre done you will press OK

4. Once the window disappears, click on the Close & Apply button in the ribbon to apply the changes. 

5. Once back to the Report View screen, right click on our Measure table and click New Measure. 

6. Once clicked a new column under our table will appear and a type of box will appear under the ribbon. In the type of box, we will enter our DAX formula in order to create our measure. Once finished typing the DAX Formula press enter and you will now have your new measure.
7. Example of what the finished product should look like.


8. Repeat steps 5 7 for all measures. Once finished you should have around 30 measures. I will provide the DAX Formulas for all the measures towards the end of the instructions.
9. Once you have finished all your measures, you should be able to delete the blank column that is within the Measure table. In order to do this, you will have to hover over the Measure table and find Column 1. Right click the column and press the Delete from Model button. Since there is nothing in the column itself it should not affect your work if you already have other measures in place.

Measures:
  • Total Orders = SUM(OrdersandDeliveries[Order Total])
  • Total Deliveries = SUM(OrdersandDeliveries[Delivery Total])
  • Orders Per Month = AVERAGEX( VALUES('OrdersandDeliveries'[Order Date]), CALCULATE(SUM('OrdersandDeliveries'[Order Total])))
  • Average Delivery = AVERAGEX(VALUES('OrdersandDeliveries'[Order Date]), CALCULATE(SUM('OrdersandDeliveries'[Delivery Total])))
  • Orders with No Deliveries = CALCULATE([Orders Per Month] - [Average Delivery])
  • Orders LY = CALCULATE([Total Orders], SAMEPERIODLASTYEAR('Calendar'[Date]))
  • Deliveries LY = CALCULATE(SUM(OrdersandDeliveries[Delivery Total]), PREVIOUSYEAR('Calendar'[Date]))
  • Orders vs Deliveries = CALCULATE([Total Orders] - [Total Deliveries])
  • Orders vs Deliveries Var = DIVIDE([Orders vs Deliveries], [Total Orders])
  • Current Year Order = TOTALYTD(SUM(OrdersandDeliveries[Order Total]),OrdersandDeliveries[Order Date])
  • Current Month Order = TOTALMTD(SUM(OrdersandDeliveries[Order Total]),OrdersandDeliveries[Order Date])
  • Average Unfilled Orders = AVERAGEX(VALUES('OrdersandDeliveries'[Unfilled Orders]),CALCULATE(SUM('OrdersandDeliveries'[Unfilled Orders])))
  • Delivery % = DIVIDE([Total Deliveries], [Total Orders])
  • .Arrow YOY = SWITCH(TRUE(), [Order YoY % Change] < 0, UNICHAR(9660), UNICHAR(9650))
  • Delivery % Arrow = SWITCH(TRUE(), [Delivery %] < 0, UNICHAR(9660), UNICHAR(9650))
  • Delivery YoY % Arrow = SWITCH(TRUE(), [Delivery YOY %] < 0, UNICHAR(9660), UNICHAR(9650))
  • Order YoY Arrow = SWITCH(TRUE(), [Order YoY %] < 0, UNICHAR(9660), UNICHAR(9650))
  • Delivery YOY % = VAR __PREV_YEAR = CALCULATE([Total Deliveries],DATEADD('Calendar'[Date],-1, YEAR))
                         RETURN
 
                                DIVIDE([Total Deliveries]- __PREV_YEAR, __PREV_YEAR)
  • Delivery YoY % Change = DIVIDE(( [Total Deliveries] - [Deliveries LY]), [Deliveries LY], 0)
  • Delivery YOY % Change (2) = VAR __PREV_YEAR = CALCULATE([Average Delivery],DATEADD('Calendar'[Date],-1, YEAR))
                         RETURN
 
                                DIVIDE([Average Delivery]- __PREV_YEAR, __PREV_YEAR)
  • Delivery % Color = IF([Delivery %]>0,"#1c9407", "#c4342d")
  • Delivery YoY % Color = IF([Delivery YOY %]>0,"#1c9407", "#c4342d")
  • Order YoY % = VAR __PREV_YEAR = CALCULATE([Total Orders],DATEADD('Calendar'[Date],-1, YEAR))
                         RETURN
 
                                DIVIDE([Total Orders]- __PREV_YEAR, __PREV_YEAR)
  • Order YoY % Change = (DIVIDE([Total Orders], [Orders LY],0)-1)
  • Order YoY % Change (2) = VAR __PREV_YEAR = CALCULATE([Orders Per Month],DATEADD('Calendar'[Date],-1, YEAR))
                         RETURN
 
                                DIVIDE([Orders Per Month]- __PREV_YEAR, __PREV_YEAR)
  • Order YoY % Color = IF([Order YoY %]>0,"#1c9407", "#c4342d")
  • Order MoM % = VAR __PREV_MONTH = CALCULATE([Total Orders],DATEADD('Calendar'[Date],-1, MONTH))
                         RETURN
 
                                DIVIDE([Total Orders]- __PREV_MONTH, __PREV_MONTH)
  • Order MoM % Color = IF([Order MoM %]>0,"#1c9407", "#c4342d ")
  • Delivery MoM % = VAR __PREV_MONTH = CALCULATE([Total Deliveries],DATEADD('Calendar'[Date],-1, MONTH))
                         RETURN
 
                                DIVIDE([Total Deliveries]- __PREV_MONTH, __PREV_MONTH)
  • Delivery MoM % Color = IF([Delivery MoM %]>0,"#1c9407", "#c4342d ")
