Transforming the Data (Order Date) 8
Once data has been successfully loaded into Power BI, we will have to tweak the data in order for us to get accurate visuals.

1. Click Transform Data in the Home category of the ribbon.


2. Once selected, a window will appear of the dataset within the Power BI Desktop Software. This will be where changes to the data occur in order for the dataset to run smoothly within Power BI.


3. First, we have to merge the columns Order Year, Order Month & Order Date. While holding the CTRL button on the keyboard select all 3 columns at once.
4. Once highlighted, you will right click and select the Merge Columns option in the dropdown.


5. A window will appear once you press the merge button to rename the new column we are creating. Since there is an Order Date column already in the dataset, we will rename the column to Order Date 1. (we will change the name later)


6. Once Renamed all three columns should be merged together, we will then rename the column again, deleting the number 1 from the title. 


7. After renaming the merged column, we must change the data type back to date since it changed when we merged the columns together. Right click Order Date. In the dropdown you will click Change Type -> Date.  The format should now be changed and able to be used in the project.
