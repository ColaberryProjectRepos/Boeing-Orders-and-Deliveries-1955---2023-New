Creating Calendar Table 11



1. Open the Data View on the left side of the desktop, it should be right under Report View 


2. Once open we will create a calendar table to be able to link the tables together. Click New Table in the Home tab in the ribbon. 



3. Once pressed a box will appear to enter a DAX Formula in order to create the table. Paste this DAX Formula in the box to create the Calendar Table. 
      Calendar = ADDCOLUMNS
            (CALENDAR 
             (MIN('OrdersandDeliveries'[Order Date]),
            MAX ( 'OrdersandDeliveries'[Order Date] )),
            "Year", YEAR([Date]),
           "Month", Format ([Date], "mmm"),
            "Month Number", MONTH([Date]),
           "Day", DAY([Date]))
     
Once you paste this formula into the box press ENTER and the table will be created under the name Calendar.
